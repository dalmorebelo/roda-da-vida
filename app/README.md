# App Frontend - Roda da Vida

Este diretório contém o código-fonte da aplicação React.

## Configuração Técnica

- **Framework**: React 19 (Vite)
- **Estilização**: CSS Nativo / CSS Modules
- **Estado**: Hooks Customizados + LocalStorage

Para informações completas sobre o projeto, arquitetura e objetivos, consulte o [README principal na raiz do repositório](../README.md).

## Scripts Disponíveis

- `npm run dev`: Inicia o servidor local.
- `npm run build`: Gera o build de produção.
- `npm run lint`: Executa a verificação estática de código.
